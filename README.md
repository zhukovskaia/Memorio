# 🐍 PythonMemorio

Progressive Web Application для изучения терминологии и основ Python через игровые механики. Превращает заучивание определений в увлекательный процесс с системой уровней, статистикой и адаптивной сложностью.

## 📌 Описание
Приложение ориентировано на студентов и начинающих разработчиков, изучающих Python. Три игровых режима позволяют отрабатывать теорию, решать задачи на вывод кода и проверять скорость реакции. Все данные хранятся локально, прогресс синхронизируется с аккаунтом пользователя.

## ✨ Ключевые возможности
- 🔐 **Авторизация и профиль** — регистрация, вход, безопасные сессии
-  **Режим «Карточки»** — свободное изучение без таймеров и очков. Клик → ответ. Доступны все 5 уровней сложности.
- ❓ **Режим «Викторина»** — 10 вопросов за раунд. Смешивает теорию и практические задачи на вывод кода. 5 уровней сложности.
- ⚡ **Режим «Спринт»** — 60 секунд на максимум баллов. Система комбо: 5 верных ответов подряд дают множитель `x2`. Выбор сложности перед стартом.
-  **Прогрессия** — уровни игр (XP), уровень аккаунта (растёт каждые 150 баллов), глобальная статистика (игры, точность, процент успеха).
- 🔔 **Уведомления** — базовая система напоминаний о повторении сложных тем.
-  **PWA** — установка на телефон/десктоп, работа в режиме `standalone`, адаптивный интерфейс.

##  Стек технологий
| Слой | Технологии |
|------|------------|
| **Backend** | Python 3, Flask, Jinja2 |
| **Frontend** | HTML5, CSS3, Vanilla JavaScript (Fetch API) |
| **Данные** | JSON-хранилище (`users_data.json`, `words_data.json`, `notifications.json`) |
| **PWA** | `manifest.json`, набор иконок, viewport-оптимизация |
| **Безопасность** | Flask-сессии, хеширование паролей (SHA-256) |
